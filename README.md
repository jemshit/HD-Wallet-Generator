#### Build 'libsecp256k1'

```shell
# on macOS only
brew install autoconf automake libtool

cd src/contrib

# might be necessary
chmod a+x make_libsecp256k1.sh

# build libsecp256k1 on host machine (architecture dependent) 
./make_libsecp256k1.sh
```
